# Data Structures & Algorithms

## Array

- 여러 data가 메모리 덩어리 안에 줄줄이 나열된 가장 간단한 구조
- data에는 index로 접근

### 시간복잡도

* Average
    * 검색 - O(n)
    * 삽입 - O(n)
    * 삭제 - O(n)
* Worst
    * 검색 - O(n)
    * 삽입 - O(n)
    * 삭제 - O(n)

## Stack

### 시간복잡도
