# Data Structures & Algorithms

## Array

- 여러 data가 메모리 덩어리 안에 줄줄이 나열된 가장 간단한 구조
- data에는 index로 접근

### 시간복잡도

* Average
    * 검색 - O(n)
    * 삽입 - O(n)
    * 삭제 - O(n)
* Worst
    * 검색 - O(n)
    * 삽입 - O(n)
    * 삭제 - O(n)

## Stack

- First In, Last Out

### 시간복잡도

* Average
    * 검색 - O(n)
    * 삽입 - O(1)
    * 삭제 - O(1)
* Worst
    * 검색 - O(n)
    * 삽입 - O(1)
    * 삭제 - O(1)

## Queue

- First In, First Out

### 시간복잡도

* Average
    * 검색 - O(n)
    * 삽입 - O(1)
    * 삭제 - O(1)
* Worst
    * 검색 - O(n)
    * 삽입 - O(1)
    * 삭제 - O(1)

## Linked List

- 메모리에서 연속되어 있지 않음. 각 노드가 포인터로 연결되어 있음. 

### 시간복잡도

* Average
    * 검색 - O(n)
    * 삽입 - O(1)
    * 삭제 - O(1)
* Worst
    * 검색 - O(n)
    * 삽입 - O(1)
    * 삭제 - O(1)
